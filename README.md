The **Smart Electricity Usage App** is an integrated mobile application designed to help users monitor, analyze, and optimize household electricity consumption. It provides intelligent calculations based on user-defined budgets and registered electrical devices, helping users save energy and reduce electricity costs. The app supports multiple input methods, including manual entry, image recognition, and voice input, making it accessible to all user types. Additionally, it includes a geolocation-based feature to find the user's regional electricity provider.
